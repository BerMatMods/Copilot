# Contributing Guidelines

Thank you for your interest in contributing to the project! Here are some guidelines to help you get started:

## Fork the Repository
1. Navigate to the [repository](https://github.com/BerMatMods/Copilot).
2. Click the 'Fork' button at the top right of the page. This will create a copy of the repository under your GitHub account.

## Create a New Branch
1. Clone your forked repository:
   ```
   git clone https://github.com/YOUR_USERNAME/Copilot.git
   cd Copilot
   ```
2. Create a new branch for your changes:
   ```
   git checkout -b your-branch-name
   ```

## Make Commits
1. Make the necessary changes to the code.
2. Stage your changes:
   ```
   git add .
   ```
3. Commit your changes with a descriptive message:
   ```
   git commit -m "Add a descriptive message here"
   ```

## Push Changes
1. Push your changes to your forked repository:
   ```
   git push origin your-branch-name
   ```

## Submit a Pull Request
1. Go to the original repository where you want to submit your changes.
2. Click on the 'Pull requests' tab.
3. Click the 'New pull request' button.
4. Select the branch you created from your forked repository.
5. Click 'Create pull request' and provide a description of your changes.

## Additional Notes
- Please ensure that your code complies with the project’s style guidelines.
- It’s a good idea to open an issue first to discuss your proposed changes with the maintainers.

Thank you for contributing!
